# Ejderha Zindanı - Gyro Sensör Oyunu

Telefonunuzun hareket sensörlerini kullanarak oynanan interaktif bir macera oyunu.

## Kurulum

1. Projeyi klonlayın
2. `npm install` komutu ile bağımlılıkları yükleyin
3. `npm start` ile geliştirme sunucusunu başlatın
4. `npm run deploy` ile GitHub Pages'e yayınlayın

## Nasıl Oynanır

- Telefonunuzu bir kılıç gibi kullanın
- Adımlarla ilerleyin
- Ejderhayı yenin!

## Gereksinimler

- HTTPS bağlantısı
- Mobil cihaz (iOS Safari veya Android Chrome)
- Gyroscope ve Accelerometer sensörleri
